## 功能说明
如果返回结果中 `code` 字段不等于"0"，则表示调用 API 接口失败。例如：
```json
{
  "code": "100001",
  "message": "用户不存在"
}
```
上述返回中 `code` 表示错误码，`message` 表示该错误的具体信息。每个错误同时也对应一个 HTTP 状态码，比如上述错误码对应了 HTTP 状态码 404(Internal Server Error)。
## 错误码列表
系统支持的错误码列表如下：

| HTTP Code |  Code | Description |
| --------- |  ---- | ----------- |
| 404 | 100001 | 用户不存在 |
